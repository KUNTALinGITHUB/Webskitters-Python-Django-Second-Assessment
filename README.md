# Webskitters-Python-Django-Second-Assessment
Date: 27-04-2026 Full Marks: 100 Duration: 3 Hours

Q2. Student Registration &amp; Management System
You are required to develop a basic Django web application to manage student
registrations.
Requirements:
1. Project Setup
• Create a Django project and a single application named students.
• Include comments explaining the purpose of key files.
2. Data Model
Design a data model to store:
• Student Name
• Email Address
• Course Name
• Marks
Ensure the data is displayed meaningfully wherever required.
3. Application Logic
• Implement one page to add new student records.
• Implement one page to display all student records.
• Application should support:
• Data submission
• Data retrieval
• Page redirection after successful actions

4. URL Handling
• Configure clean and readable URL patterns.
• Use dynamic routing where necessary.

5. User Interface
• Display student records in a tabular format.
• Use conditional rendering where required.
• Ensure common layout is reused across pages.
6. Concept Explanation
Add comments in the code to explain:
• How request handling flows through the application
• How different request types are handled differently
